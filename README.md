# 全球热点新闻速览

这是一个纯前端的新闻聚合仪表盘，帮助你在每天开始时快速浏览海外最新、最热的资讯。页面会调用 [GNews API](https://gnews.io/) 获取国际新闻，并根据标题与内容生成简短摘要，支持主题、语言、地区以及关键词检索等筛选。

> **温馨提示**：GNews API 需要注册免费账号并获取 API Key。没有填写 API Key 时，页面会展示一组内置的示例数据，方便预览界面效果。

## 功能亮点

- ⏱️ 实时显示当前时间，方便掌握查看时间点。
- 🌍 一键切换主题、语言、地区，聚焦你关心的海外资讯。
- 🧠 自动提取新闻描述，生成 1-2 句摘要，快速了解重点。
- 🔍 支持关键词搜索，随时追踪感兴趣的话题。
- 💾 设置持久化保存在浏览器本地存储，下次打开直接生效。

## 使用步骤

1. 访问 `index.html`，在左侧“数据源设置”中填入你的 GNews API Key。
2. 根据需要调整主题、语言、地区和返回数量。
3. 点击“保存设置”以便下次自动加载，或点击“立即更新”获取最新新闻。
4. 通过搜索框输入关键词并回车，即可检索相关报道。

## 自定义与部署

- 如需集成其他新闻源，可在 `news.js` 中修改 `buildUrl` 函数的接口地址与参数。
- 样式定义集中在 `style.css`，可根据品牌或个人喜好进一步定制主题色和布局。
- 这是一个静态站点，直接部署到 GitHub Pages、Vercel、Netlify 等平台即可上线使用。
